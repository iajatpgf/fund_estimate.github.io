# 📈 基金估值 — 手机端自选基金实时估值查看工具

基于天天基金公开数据，在手机上查看自选基金的实时估值、涨跌幅和持仓收益。

## ✨ 功能

- 📊 **实时估值** — 交易时段实时更新基金估算净值
- 💰 **持仓收益** — 设置份额和成本价，自动计算持仓盈亏
- 🔍 **基金搜索** — 支持代码/名称/拼音搜索，快速添加自选
- 🌓 **深色模式** — 自动跟随系统或手动切换
- 📱 **PWA支持** — 添加到手机主屏幕，像原生App一样使用
- 💾 **本地存储** — 数据存在手机本地，无需注册账号
- 📤 **导入导出** — JSON格式备份和恢复基金配置

## 🚀 快速开始

### 方法一：Python代理服务器（推荐，零配置手机访问）

```bash
# 1. 启动代理服务器
python proxy.py

# 2. 手机连接同一WiFi，浏览器打开终端显示的地址
#    例如: http://192.168.1.100:8888

# 3. 添加到手机主屏幕即可
```

### 方法二：部署到公网（Netlify，免费）

1. Fork或上传本项目到GitHub
2. 在 [Netlify](https://app.netlify.com) 导入项目
3. 部署完成后，在App设置中将代理地址设为：`/api/fundgz/{code}`
4. 手机浏览器打开Netlify提供的域名，添加到主屏幕

### 方法三：直接打开（仅限电脑本地预览）

直接在浏览器中打开 `index.html` 文件，通过JSONP方式获取数据。

> ⚠️ 手机上直接打开HTML文件可能无法正常工作，建议使用方法一或方法二。

## 📖 使用说明

1. 点击右下角 **+** 按钮
2. 输入基金代码（如 `161725`）或名称搜索
3. 点击添加
4. 点击基金卡片展开持仓设置
5. 输入持有份额和成本价，保存后自动计算盈亏

## 🔧 代理设置

如果部署到HTTPS环境（如Netlify/GitHub Pages），需要在设置中配置代理地址：

| 部署方式 | 代理地址 |
|---------|---------|
| Python代理 | `http://你的IP:8888/proxy?code={code}` |
| Netlify | `/api/fundgz/{code}` |
| Vercel | `/api/fundgz/{code}` |

## 📡 数据接口

数据来自天天基金公开接口：

- 估值数据：`fundgz.1234567.com.cn/js/{基金代码}.js`
- 基金列表：`fund.eastmoney.com/js/fundcode_search.js`
