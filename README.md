# 📈 基金估值 — 手机端自选基金实时估值查看工具

使用太一数据盘中估值与东方财富官方净值，在手机上查看自选基金的实时估值、涨跌幅。

## ✨ 功能

- 📊 **实时估值** — 交易时段实时更新基金估算净值和涨幅，盘后自动切换到实际涨幅
- 🔍 **基金搜索** — 支持代码/名称/拼音搜索，快速添加自选基金
- 📊 **多维度排序** — 支持按涨幅升序/降序、按基金名称排序
- 🌓 **深色模式** — 自动跟随系统或手动切换
- 📱 **PWA 支持** — 添加到手机主屏幕，像原生 App 一样使用
- 💾 **本地存储** — 数据存在手机本地，无需注册账号
- 📤 **导入导出** — JSON 格式备份和恢复基金配置，兼容自选基金助手格式
- 🔄 **自动刷新** — 交易时段每 30 秒自动刷新估值（可开关）
- 🏦 **多数据源容错** — 太一数据盘中估值 → 东方财富官方净值，多层容错

## 🚀 快速开始

### 方法一：Python 代理服务器（推荐，零配置手机访问）

```bash
# 1. 启动代理服务器
python proxy.py

# 2. 手机连接同一 WiFi，浏览器打开终端显示的地址
#    例如: http://192.168.1.100:8888

# 3. 添加到手机主屏幕即可
```

### 方法二：部署到公网（Netlify / Vercel，免费）

1. Fork 或上传本项目到 GitHub
2. 在 [Netlify](https://app.netlify.com) 或 [Vercel](https://vercel.com) 导入项目
3. 部署完成后，手机浏览器打开分配域名，添加到主屏幕

### 方法三：直接打开（仅限电脑本地预览）

直接在浏览器中打开 `index.html` 文件。

> ⚠️ 手机上直接打开 HTML 文件可能无法正常工作，建议使用方法一或方法二。

## 📖 使用说明

1. 点击底部 **➕ 添加** 按钮
2. 输入基金代码（如 `161725`）或名称搜索
3. 点击添加
4. 点击基金卡片展开详情，查看估算涨幅、估算净值、实际涨幅和单位净值
5. 点击 **📊 排序** 切换排序方式（涨幅降序 → 涨幅升序 → 名称升序 → 名称降序 → 默认）
6. 点击 **⚙️ 设置** 配置太一数据 API Key、深色模式、自动刷新、代理地址或导入导出数据

## 🔑 太一数据盘中估值

1. 在太一数据申请 API Key。
2. 打开应用 **设置**，在“太一数据 API Key”填入 `sk-...`。
3. 交易时段会优先通过太一数据获取分钟级估值；非交易时段接口返回空数据，应用自动展示东方财富的官方最新净值。

> API Key 会保存到当前浏览器的 localStorage，且不会包含在导出文件中。部署到公共网站时，浏览器网络请求仍会携带该 Key；如需多人使用，请改由自己的后端代理保管 Key。

## 🔧 代理设置

代理仅用于旧兼容数据源或特殊网络环境；盘中估值请优先配置太一数据 API Key。

| 部署方式 | 代理地址 |
|---------|---------|
| Python代理 | `http://你的IP:8888/proxy?code={code}` |
| Netlify | `/api/fundgz/{code}` |
| Vercel | `/api/fundgz/{code}` |

> 💡 **提示**：大部分情况下留空代理地址即可，应用会自动选择最优数据源。

## 📡 数据接口

数据来自太一数据与东方财富接口：

- 估值数据（主）：`api.wanxingai.com/openapi/get_fund_realtime_valuation/`（太一数据，需 API Key）
- 官方净值/实际涨跌：`fundmobapi.eastmoney.com/FundMNewApi/FundMNFInfo`
- 基金列表：`fund.eastmoney.com/js/fundcode_search.js`
- 官方净值：`fund.eastmoney.com/pingzhongdata/{基金代码}.js`
